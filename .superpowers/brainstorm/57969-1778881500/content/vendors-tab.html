B — Cards list + side drawer

Each vendor is a card showing name, phone, email, status. Same drawer for add/edit. More spacious, shows more info at a glance, but fewer vendors visible without scrolling.

Both use the existing Drawer component. A is denser (more vendors visible). B shows more info per vendor.
